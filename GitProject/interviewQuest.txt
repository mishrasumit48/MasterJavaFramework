public class temp
{
    public static void main(String args[])
    {
        int ok=10;
        switch(ok)
        {
            default:
                System.out.println("default");
            case 0:
                System.out.println("true");
            case 1:
                System.out.println("false");
        }
    }
}

Ans: 
default true false

Since switch falls top to bottom and there is no break statement used after default and case block, so default true false will print.

public class temp
{
    public static void main(String args[])
    {
        boolean ok=true;
        switch(ok)
        {
            case true:
                System.out.println("true");
            break;
            case false:
                System.out.println("false");
            break;
            default:
                System.out.println("default");
            break;
        }       
    }
}

Ans : Compilation time error "incompatible types" will occur, we cannot use boolean value with switch statement.
